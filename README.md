# stylish-song-project

This project is a simple HTML and CSS webpage displaying the lyrics of the song "Yesterday" by The Beatles.

## Files

- index.html - Contains the webpage structure and lyrics.
- style.css - Contains the styling, colors, gradients, shadows, and formatting.
